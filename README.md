# hello-world
Criando uma aplicação “Hello World” em Spring Boot 2

### Detalhes da API Hello World
* Projeto criado com Spring Boot 2 e Java 8

### Como executar a aplicação
```
git clone https://github.com/andrecostan/hello-world.git
cd hello-world
mvn spring-boot:run
```
